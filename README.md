# Avanak
## پیام صوتی آوانک
آوانک اولین اپراتور پیام صوتی در ایران می باشد که با گستردگی زیر ساخت های مخابراتی توانایی برقراری تماس و ارسال صدای شما به صورت همزمان برای انبوهی از مخاطبین تلفن ثابت و همراه را فراهم کرده است.
